# Expense Tracker

A privacy-first, browser-based expense tracker with categories, monthly reporting, and CSV export.

## Features
- Add/edit/delete expenses
- Categories (preset + custom)
- Monthly chart + budget tracking
- CSV export
- All data stored locally (no accounts required)

## Tech
- React 18
- Recharts for charts
- Vite for build
- Netlify for deployment

## Getting Started
1. Clone the repo
2. `npm install`
3. `npm run dev`
4. Open http://localhost:5173

## Deployment
Deployed on Netlify. Auto-deploys on push to main.

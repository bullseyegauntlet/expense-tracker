## 🔧 Manual Setup Steps (if needed)

To enable **continuous deployment** from GitHub to Netlify:

1. Navigate to https://app.netlify.com/projects/expense-tracker-gauntlet/configuration/deploys
2. Click "Link repository" → "Link to an existing repository..." → GitHub
3. Authorize Netlify with GitHub
4. Select `bullseyegauntlet/expense-tracker`
5. Configure build settings:
   - Build command: `npm run build`
   - Publish directory: `dist`
   - Base directory: (leave blank)
6. Deploy

Once linked, any push to `main` will trigger an automatic deploy.
